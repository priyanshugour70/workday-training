### Day 0 – Resources (Read This First, Then Decide If You Need External Links)

Day 0 is designed so that you can **learn a lot just by reading inside the repo**.  
These resource notes are here only if you want to go even deeper later.

---

#### 1. Internal Resources (Inside This Repo)

- `theory.md`
  - Treat it like a **chaptered book**:
    - Role overview.
    - Workday ecosystem map.
    - Finance fundamentals with Workday view.
    - Tools (SQL, Power BI, Office) and why they matter.
- `practical-exercises.md`
  - Use exercises + model answers to **train your thinking style**.
- `qa-and-scenarios.md`
  - Many additional **question/answer pairs** and stories you can reuse in interviews.
- `glossary.md`
  - Quick place to check any unknown term before searching the internet.
- `self-assessment-quiz.md`
  - Check if you are at the “I can explain this to someone else” level.

If you are just starting, **stay inside these files first**. Only when you feel gaps, go to external resources.

---

#### 2. When and How to Use External Resources

Use external materials **with a goal**, not randomly:

- “I want concrete examples of Workday Financials screens and navigation.”
- “I want deeper knowledge of Adaptive Planning models.”
- “I want more SQL and Power BI practice problems.”

For each goal:

- Search for **recent** content (include the year, e.g. `2026`) so screenshots and features are current.
- Prefer:
  - Official product overviews (Workday marketing/overview pages).
  - Vendor documentation.
  - Reputable blogs or conference talks.

You don’t need to copy links here; just track favorites in your own notes.
